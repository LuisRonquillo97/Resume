# Luis Ronquillo Portfolio — Hybrid Punch

Versión híbrida:
- Mantiene mood visual tipo placeholder de Luis.
- Agrega el punch profesional para vacantes de 80k-110k+.
- Vue 3 + TypeScript + Vite.
- Bilingüe ES/EN.
- Gengar Mode oscuro morado.
- Blaziken Mode claro naranja.
- Contenido orientado a Lead Backend Engineer.

## Correr local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy recomendado

Vercel o Netlify.
